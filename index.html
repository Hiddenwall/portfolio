Good Boy VP

A University Game Project.
Good Boy VP is an AI virtual pet which the user can interact with via commands. Images and descriptions displayed to the screen allow the user to determine what state the dog is in. The pet will react to commands differently depending on its internal states, ie, wont eat or play if sick, or won't react while sleeping.
